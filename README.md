# Online Event Booking System

An online event booking web application developed using React JS. This project allows users to view events, book tickets, and manage event details through a simple and user-friendly interface.

## Features
- User-friendly interface
- View available events
- Event booking functionality
- Responsive design
- Navigation between pages using React Router
- Booking form with validation
- Dynamic event display

## Technologies Used
- React JS
- HTML5
- CSS3
- JavaScript
- React Router DOM
